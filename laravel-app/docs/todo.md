# TODO - 学校図書管理システム

## バグ・UI修正

### 1. 登録ボタンが見えない
- **状況**: 新規登録ページの「登録する」ボタンが白背景に溶けて見えない
- **原因**: `bg-primary` のカスタムカラーが本番ビルドで反映されていない
- **対象**: `resources/views/auth/register.blade.php`（他のページにも同じ問題がある可能性）

### 2. 「安全ではありません」画面が表示される
- **状況**: 登録後の画面遷移時に安全ではないという警告が出る
- **原因**: 要調査（CSRF、リダイレクト、または混合コンテンツの問題の可能性）

### 3. 書籍の重複登録の対応
- **状況**: 同じISBNの書籍が複数登録された場合の挙動が未定義
- **対応案**:
  - ISBNユニーク制約はDBにあるが、ユーザーへのエラーメッセージの改善が必要
  - 登録時に既存チェック → 「この書籍は既に登録されています」と表示

## デプロイ関連

### 4. Tailwindカスタムカラーが本番で効かない
- **状況**: `bg-primary` 等のカスタムカラーが本番環境で反映されない
- **原因**: Vite/Tailwindのビルドプロセスの問題（要調査）
- **影響**: ボタン、バッジ等の色が全ページで消えている可能性

- **対応案**:
  - UptimeRobot等の無料監視サービスで定期ping（5〜14分間隔）を設定してスリープを防止
  - または Render有料プラン（$7/月）に変更
