# 学校図書管理システム - システム概要

## このアプリについて
学校の図書室の蔵書管理・貸出管理を行うWebアプリケーションです。
スマートフォン・タブレット・PCからアクセスできます。

## 主な機能

### 一般ユーザー（生徒・教員）
| 機能 | 説明 |
|---|---|
| 書籍検索 | タイトル・著者名で蔵書を検索 |
| 書籍詳細 | 書籍の詳細情報（出版社・説明・表紙画像）を閲覧 |
| 貸出 | 書籍一覧から「借りる」ボタンで貸出手続き |
| 返却確認 | マイページで自分の貸出中の書籍を確認 |

### 管理者（図書担当教員）
| 機能 | 説明 |
|---|---|
| 書籍登録 | ISBN読み取り or 手動入力で新規書籍を登録 |
| ISBN自動入力 | ISBNを入力すると書籍情報を自動取得（Google Books API） |
| 貸出管理 | 全生徒の貸出状況を一覧表示 |
| 延滞検知 | 返却期限を過ぎた貸出を自動で「延滞」に変更（日次実行） |

## 技術構成

| 項目 | 技術 |
|---|---|
| フレームワーク | Laravel 12（PHP） |
| データベース | PostgreSQL（Neon） |
| ホスティング | Render |
| フロントエンド | Blade + Tailwind CSS + Alpine.js |
| 外部API | Google Books API（ISBN検索） |
